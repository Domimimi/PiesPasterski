Jak to uruchomić?

1. Zainstaluj biblioteki: pip install gymnasium stable-baselines3 pygame kazoo numpy
2. Włącz serwer Zookeeper (na porcie 2181)
4. Uruchom plik config_manager.py (wysłanie konfiguracji)
5. Uruchom plik train.py (trenowanie psa)
6. Uruchom plik test.py aby zobaczyć efekt treningu

W pliku sheepdog_final_model.zip jest zapisany wytrenowany model psa (w miarę łapie owcę).
